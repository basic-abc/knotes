### [Constraint cheat sheet](https://www.hackerearth.com/practice/basic-programming/complexity-analysis/time-and-space-complexity/tutorial/)
- n <= [10 ... 11]: O(N!)
- n <= [18 ... 22]: O(2^n * n)
- n <= 100: O(n^4)
- n <= 400: O(n^3)
- n <= 10^4: O(n^2)
- n <= 10^6: O(n*logn)
- n <= 10^8: O(n), O(nlogn)
